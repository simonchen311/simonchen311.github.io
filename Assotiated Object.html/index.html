关联对象

开发
  • iOS

定义

关联对象是指两个对象关联起来，其中一个对象作为另一个对象的一部分；它允许开发者为任何对象附着键值数据。

可以给某对象关联许多其他对象，这些对象通过“键”来区分。存储对象值的时候，可以指明“存储策略”，用以维护相应的“内存管理语义”。

设置关联对象用的键是个“不透明的指针”，即所指向的数据结构不局限于某种特定类型的指针。再设置关联对象值时，若想令两个键匹配到同一个值，则二者必须时完全相同的指针才行。故设置关联对象值时，通常使用静态全局变量做键。

下列方法可以管理关联对象：

（1）以给定的键和策略为某对象设置关联对象值

（2）根据给定的键从某对象中获取响应的关联对象值。

（3）移除指定对象的全部关联对象

参数

id object 被关联的对象

const void *key 要关联的对象的键值

id value 要关联的对象

objc_AssociationPolicy policy 关联时采用的存数策略
