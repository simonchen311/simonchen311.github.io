多GitHub账号的SSH配置

开发
  • git

github上使用ssh key可以不用每次都输入用户名密码，一定程度上可以提高效率。

如果是单账户，很方便，默认拿id_rsa与你的 github服务器的公钥对比；如果是多用户如user1,user2,那么就不能用在user2的身上了，怎么办呢？

下面详细说明下多github用户配置ssh key。


创建key

利用ssh-keygen -t rsa -C "[email]"创建不同的ssh key，但是不能再一路回车了，需要给文件创建别名，以防止被覆盖：

Enter file in which to save the key (/Users/Sunghao/.ssh/id_rsa): [输入任意的rsa名称]
