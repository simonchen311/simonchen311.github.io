SDWebImage源码阅读

开发
  • Source Code
  • iOS

SDWebImage这套开源库可以说是开发必备品了，于是写这篇文章对其内部主要源码的实现好好理顺一下。

UIImageView+WebCache

我们在使用SDWebImage时，经常是利用它提供的接口直接在UIImageView或者UIButton上设置URL来下载图片并且显示出来

以UIImageView+WebCache为例，其内部实际上是一个深层次的调用

UIView+WebCache

继续查看，可以发现sd_internalSetImageWithURL:placeholderImage:options:operationKey:setImageBlock:progress:completed:是UIView分类提供的方法，在这个方法里面，主要做了这么些事情：

  1. 通过私有方法sd_cancelImageLoadOperationWithKey:取消当前UIView上正在下载的图片
  2. 设置占位图片
  3. 通过SDWebImageManager来下载对应的url图片，并且在图片下载完成后在主线程上设置图片到当前的UIView中显示，并且执行对应的completedBlock
