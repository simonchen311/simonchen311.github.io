JSONModel 源码阅读

开发
  • Source Code
  • iOS

JSONModel是一个解析JSON的开源库，其作用就是将JSON数据直接解析成自定义的Model。

用法

使用JSONModel非常简单，只需要将代码中定义的Model继承自JSONModel，而后调用JSONModel中的接口，让其内部自己完成从JSON数据中解析出对应Model中的属性并进行赋值工作。

源码

那么JSONModel内部解析的过程是什么样子的呢，这就是写这篇文章的主要目的：探究其实现源码。

JSONModel提供了多种接口给外部调用，我们可以直接传递NSString，NSData或者NSDictionary让JSONModel进行JSON解析，但是这些接口终究会调用到一个最终的统一接口
